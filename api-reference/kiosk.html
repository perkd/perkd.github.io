Kiosk API Reference ​

This page provides complete API documentation for Perkd's kiosk machine integration endpoints.

Create Kiosk Order ​

POST
/kiosk/order

Creates an order at a kiosk terminal for food or beverages.

Request Body​

application/json
JSON
{
"kioskId": "KIOSK-001",
"personId": "string",
"items": [
{
"itemId": "string",
"quantity": 0,
"options": [
{
"optionId": "string",
"value": "string"
}
]
}
]
}

Responses​

Order created successfully

application/json
JSON
{
"orderId": "string",
"orderNumber": "string",
"totalAmount": 0,
"estimatedWaitTime": 0,
"status": "string"
}

Responses​

Order created successfully

application/json
JSON
{
"orderId": "string",
"orderNumber": "string",
"totalAmount": 0,
"estimatedWaitTime": 0,
"status": "string"
}
